# Color Palette Reference

## 프롬프트 (팔레트 시트 생성용)

```
Create a pixel art color palette sheet for a cute idle mining game.
The palette should be pastel-toned, soft, and not too dark.
Include these color groups arranged in a grid:

- Skin tones: 3 shades (light peach)
- Hair: brown, black, blonde (3 shades each)
- Clothing sets:
  - Rags (beige/brown, 3 shades)
  - White t-shirt (off-white, 3 shades)
  - Denim (pastel blue, 3 shades)
  - Leather (warm brown, 3 shades)
  - Miner uniform (orange/yellow, 3 shades)
  - Golden armor (gold/yellow, 3 shades)
  - Royal robe (pastel purple, 3 shades)
- Tools:
  - Stone (gray, 2 shades)
  - Wood (warm brown, 2 shades)
  - Iron (cool gray, 2 shades)
  - Gold (yellow/gold, 2 shades)
  - Diamond (pastel cyan, 2 shades)
  - Mithril (pastel blue-silver, 2 shades)
  - Orihalcon (pastel green, 2 shades)
  - Dragonbone (pastel red-orange, 2 shades)
  - Starlight (pastel purple-white glow, 2 shades)
- UI: pastel blue (progress bar), gold coin yellow
- Pet: white/cream (3 shades for the dog)

Style: flat pixel art swatches, labeled, clean grid layout, white background.
Total palette: 32 colors maximum.
```
