# Character Sprite Prompts — Level 1~10

모든 프롬프트에 공통으로 붙일 prefix:
```
Pixel art, 32x32 sprite sheet, cute chibi character, 2.5 head-to-body ratio,
big round eyes, soft pastel colors, no anti-aliasing, clean pixel edges,
transparent background, Korean mobile idle RPG style.
4 frames: idle(2), mining(2).
```
