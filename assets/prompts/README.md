# Pixel Goldiger — AI Asset Generation Prompts

ChatGPT (DALL-E 3) 또는 Midjourney에 사용할 프롬프트 모음.
각 파일별로 프롬프트를 복사해서 사용하면 됩니다.

## 사용 순서
1. `palette.md` — 팔레트 레퍼런스 먼저 생성
2. `character-levels.md` — 레벨별 캐릭터 스프라이트
3. `pet-dog.md` — 하얀 강아지 펫
4. `backgrounds.md` — 배경
5. `ui-elements.md` — UI 요소 (골드 아이콘, 프로그래스바 등)

## 공통 스타일 키워드
모든 프롬프트에 아래 키워드를 포함할 것:
```
pixel art, 32x32, cute chibi style, pastel color palette,
clean edges, no anti-aliasing, transparent background,
Korean mobile idle RPG style (reference: 슬레이어키우기)
```

## 보정 가이드
- Aseprite에서 32x32로 리사이즈 (Nearest Neighbor)
- palette.md의 색상으로 통일
- 프레임간 1~2px만 변화시켜 자연스러운 애니메이션
