# Implementation Plan

## Phase 1: 프로젝트 세팅
- Next.js + TypeScript 프로젝트 초기화
- Phaser.js 설치 및 Next.js 통합
- 환경변수 설정 구조 (EXP 테이블, 채굴 속도 등)

## Phase 2: 게임 코어
- 게임 상태 관리 (레벨, EXP, 골드)
- EXP/레벨업 시스템 (지수 곡선, 초반 빠른 레벨업)
- 채굴 타이머 (레벨별 속도 차등)
- 환경변수 기반 밸런스 조정 구조

## Phase 3: 픽셀 아트 & 애니메이션
- 캐릭터 스프라이트 (10레벨 x 의상/도구 조합)
- 하얀색 강아지 펫 스프라이트 + 따라다니기 로직
- 채굴 모션 애니메이션
- 배경 (파스텔 톤, 프로토타입 2~3종)

## Phase 4: UI
- 상단 골드 표시
- 하단 파란 EXP 프로그래스바
- 레벨업 이펙트 (반짝임)

## Phase 5: 방치 시스템
- 오프라인 보상 계산
- 복귀 시 보상 팝업
