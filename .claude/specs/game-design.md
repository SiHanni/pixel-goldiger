# Game Design Spec

## Concept
- **장르**: 방치형 (Idle/Incremental) 픽셀 아트 게임
- **컨셉**: 광부키우기 — 광산을 파며 캐릭터를 성장시키는 방치형 게임
- **플랫폼**: Flutter 앱 내 Next.js WebView
- **아트 스타일**: 귀여운 픽셀 아트, 전연령대 남녀노소 타겟

## Benchmarking References
- [슬레이어 키우기 (GEAR2)](https://apps.apple.com/kr/app/%EC%8A%AC%EB%A0%88%EC%9D%B4%EC%96%B4-%ED%82%A4%EC%9A%B0%EA%B8%B0/id1635712706) — 메인 벤치마킹 대상
  - 참고 포인트: 픽셀 아트 스타일, 방치 보상 시스템, 캐릭터 성장 루프, UI/UX 구성
  - 퀄리티 기준: 슬레이어키우기 수준의 픽셀 에셋 퀄리티

## Background / Environment
- **톤**: 파스텔 톤 (너무 어둡지 않은)
- **계절**: 봄, 여름, 가을, 겨울
- **시간대**: 오전(해가 뜨는), 오후(해가 지는)
- **장소**: 숲, 바다 등 다양한 배경
- 계절 x 시간대 x 장소 조합으로 배경 다양성 제공
- 프로토타입: 레벨 기반 배경 전환 (Lv1~3 숲, Lv4~6 동굴, Lv7~9 바다 절벽, Lv10 하늘 광산)

## Pet System
- **기본 펫**: 하얀색 강아지
- 캐릭터를 따라다니는 동반자
- 귀여운 idle 애니메이션 (앉기, 꼬리 흔들기 등)

## Mining Speed (per level)
- 레벨이 올라갈수록 채굴 속도가 아주 조금씩 빨라짐
- 속도 값은 환경변수로 조정 가능하게 구현

## EXP System
- 경험치 테이블은 환경변수로 조정 가능
- 레벨업에 필요한 경험치는 레벨이 올라갈수록 증가 (지수 곡선)
- 초반 레벨업은 빠르게 → 리텐션 확보

## UI Layout (슬레이어 키우기 참고)
- **상단**: 현재 보유 골드 표시
- **중앙**: 캐릭터 + 펫(하얀 강아지) + 채굴 애니메이션 + 배경
- **하단**: 파란색 프로그래스바 (경험치)
- 레벨업 시 시각적 이펙트 (반짝임 등)

## Core Loop
1. 광석 채굴 (자원 수집) — 레벨에 따라 속도 증가
2. 경험치 획득 → 레벨업
3. 레벨업 시 외형(의상+도구) 자동 변경
4. 더 빠른 채굴 → 더 많은 골드
5. 방치 중에도 자원 획득 (오프라인 보상)
